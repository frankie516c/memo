<h3> 만능버튼의 오류 보완 </h3>

`var state = false;`: 전역변수<br />
    전체를 control하는 것이기 때문에 얘가 중요한 포인트가 됨<br />
`load`함수를 부를것이기 때문에<br />
`state = !state;` 얘가 없어져야 함<br />

그리고 `load`랑 `clean` 함수에 `state`을 넣어줌<br />

study.js
```
console.log("스터디01")
function load() {
var html = "";
for(let i = 0; i < 25; i++ ) {
    let css = "1";
    if(i % 2 === 0) css = "2"
        html += `<div class="bg${ css }"></div>`;
}
document.getElementsByTagName("section")[0].innerHTML = html;
state = true;
}

function clean() {
    document.getElementsByTagName("section")[0].innerHTML = "";
    state = false;
}
var state = false; 
function btnToggle() {
    if(!state) load();
    else clean();
}
```

<h3> 나머지 반 </h3>

아래 식이 갖는 의미를 설명하는것 부터 시작
```
console.log("스터디01")
function load() {
var html = "";
for(let i = 0; i < 25; i++ ) {
    let css = "1";
    if(i % 2 === 0) css = "2"
        html += `<div class="bg${ css }"></div>`;
}
document.getElementsByTagName("section")[0].innerHTML = html;
state = true;
}
```
- document: html이라는 문서
- section: html에서 가져오려고 하는 태그
- innerHTML: HTML안에서 가져오는 함수
- =html: html을 가져온다

아주 좋아. 이건 **“외워서 쓰는 코드” → “구조를 이해하는 코드”**로 넘어가는 딱 좋은 질문이야.
지금부터는 **한 단어씩, 역할 중심으로** 설명할게.
(정확한 비유 + 실제 의미 둘 다 같이 갈게)

---

## 전체 코드부터 한 번 보자

```js
document.getElementsByTagName("section")[0].innerHTML = html;
```

이 한 줄은 말로 하면:

> **HTML 문서(document) 안에서
> section 태그들을 찾고
> 그중 첫 번째 section을 골라서
> 그 안의 내용을 html 변수로 바꿔라**

야.

---

## 이제 **왼쪽부터 하나씩 분해**해보자

---

## 1️⃣ `document`

### ✔ 실제 의미

* **지금 열려 있는 웹페이지 전체(HTML 문서)**

### ✔ 비유

* 📄 하나의 “보고서 파일”
* 🏠 하나의 “집 전체”

👉 HTML에서 `<html>...</html>` 전체를 대표하는 객체

---

## 2️⃣ `.getElementsByTagName("section")`

### ✔ 실제 의미

* document 안에서
  **모든 `<section>` 태그를 찾아라**

### ✔ 중요한 포인트

* **여러 개를 한 번에 찾음**
* 결과는 **배열 비슷한 것(HTMLCollection)**

```js
document.getElementsByTagName("section")
```

👉 예를 들면 HTML이 이렇다면:

```html
<section>첫 번째</section>
<section>두 번째</section>
```

👉 결과는 내부적으로 이런 느낌:

```js
[
  <section>첫 번째</section>,
  <section>두 번째</section>
]
```

---

## 3️⃣ `("section")`

### ✔ 실제 의미

* 찾고 싶은 태그 이름

### ✔ 주의

```js
"section"  // 태그 이름만, 꺾쇠 <> 없음
```

❌ `"<section>"` 아님
❌ `"#section"` 아님

---

## 4️⃣ `[0]`

### ✔ 실제 의미

* 찾은 section들 중 **첫 번째 것**

### ✔ 이유

* JS에서 배열은 **0부터 시작**

```js
[0] → 첫 번째
[1] → 두 번째
```

👉 즉,

```js
document.getElementsByTagName("section")[0]
```

= **첫 번째 `<section>` 태그 하나**

---

## 5️⃣ `.innerHTML`

### ✔ 실제 의미

* 해당 태그 **안쪽의 HTML 내용**

### ✔ 예시

HTML:

```html
<section>
  <p>Hello</p>
</section>
```

```js
section.innerHTML
```

👉 결과:

```html
<p>Hello</p>
```

---

## 6️⃣ `= html`

### ✔ 실제 의미

* 오른쪽 값을 왼쪽에 **넣어라(대입)**

### ✔ 여기서 `html`은?

```js
let html = "<p>새 내용</p>";
```

👉 즉, section 안에 있는 내용을
**이 문자열로 바꾼다**

---

## 🔥 전체를 자연어로 다시 말하면

> **현재 웹페이지(document)에서
> 모든 section 태그를 찾고,
> 그중 첫 번째 section을 선택해서,
> 그 안의 HTML 내용을
> html이라는 변수에 들어 있는 값으로 바꾼다**

---

## 🧠 구조를 한 번에 이해하는 핵심 포인트

이 코드는 사실 **4단계 접근**이야:

1️⃣ **어디서?** → `document`
2️⃣ **무엇을?** → `section` 태그
3️⃣ **어느 것?** → `[0]`
4️⃣ **뭘 할 건데?** → `innerHTML = ...`

---

## 🚨 초보자가 제일 많이 헷갈리는 지점

### ❌ 오해

* document = HTML 태그
* getElementsByTagName = 하나만 가져온다

### ✅ 정정

* document = **페이지 전체**
* getElementsByTagName = **여러 개**
* `[0]`을 써야 하나를 집음

---

## ✍️ 너한테 추천하는 연습 (이게 진짜 중요)

### 1️⃣ 콘솔에서 이것부터 쳐봐

```js
document.getElementsByTagName("section")
```

👉 뭐가 나오는지 직접 보기

---

### 2️⃣ 그다음

```js
document.getElementsByTagName("section")[0]
```

👉 HTML 요소 하나가 보일 거야

---

### 3️⃣ 그다음

```js
document.getElementsByTagName("section")[0].innerHTML
```

👉 안에 뭐 들었는지 보기

---

### 4️⃣ 마지막

```js
document.getElementsByTagName("section")[0].innerHTML = "바뀜!"
```

👉 화면이 바뀌는 걸 직접 확인

---

## 마지막으로 아주 중요한 한 줄

> **DOM 코드는 “외우는 문장”이 아니라
> “위치 → 대상 → 행동” 구조다.**
