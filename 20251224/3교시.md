<h3> 만능버튼의 오류 보완 </h3>

`var state = false;`: 전역변수<br />
    전체를 control하는 것이기 때문에 얘가 중요한 포인트가 됨<br />
`load`함수를 부를것이기 때문에<br />
`state = !state;` 얘가 없어져야 함<br />

그리고 `load`랑 `clean` 함수에 `state`을 넣어줌<br />

study.js
```
console.log("스터디01")
function load() {
var html = "";
for(let i = 0; i < 25; i++ ) {
    let css = "1";
    if(i % 2 === 0) css = "2"
        html += `<div class="bg${ css }"></div>`;
}
document.getElementsByTagName("section")[0].innerHTML = html;
state = true;
}

function clean() {
    document.getElementsByTagName("section")[0].innerHTML = "";
    state = false;
}
var state = false; 
function btnToggle() {
    if(!state) load();
    else clean();
}
```

<h3> 나머지 반 </h3>

아래 식이 갖는 의미를 설명하는것 부터 시작
```
console.log("스터디01")
function load() {
var html = "";
for(let i = 0; i < 25; i++ ) {
    let css = "1";
    if(i % 2 === 0) css = "2"
        html += `<div class="bg${ css }"></div>`;
}
document.getElementsByTagName("section")[0].innerHTML = html;
state = true;
}
```
- document: html이라는 문서
- section: html에서 가져오려고 하는 태그
- innerHTML: HTML안에서 가져오는 함수
- =html: html을 가져온다

좋은 질문이에요 👍
말씀하신 줄은 **DOM 조작의 핵심**이라서 단어 하나하나 이해해두면 JS가 훨씬 쉬워집니다.
아래 코드를 **왼쪽 → 오른쪽**, **단어(by 단어)**로 아주 차근차근 풀어볼게요.

---

## 분석 대상 코드

```js
document.getElementsByTagName("section")[0].innerHTML = html;
```

---

## 1️⃣ `document`

### 의미

* **현재 열려 있는 웹페이지 전체(HTML 문서)**를 뜻함
* 정확히는 **브라우저가 읽고 있는 HTML 문서 객체**

📌 보통:

```html
index.html
```

이 파일 전체를 자바스크립트에서 **객체(object)** 로 접근할 때 쓰는 이름이 `document`

👉
**"지금 이 웹페이지 문서"**

---

## 2️⃣ `.` (점 연산자)

### 의미

* **document 안에 있는 기능이나 속성에 접근하겠다**

예:

```js
document.title
document.body
document.getElementById()
```

👉
**document 안에 있는 getElementsByTagName을 사용하겠다**

---

## 3️⃣ `getElementsByTagName`

### 의미

* **특정 태그 이름을 가진 모든 요소들을 가져오는 함수**
* 복수형 `Elements` 인 이유가 중요함 ❗

```js
document.getElementsByTagName("section")
```

👉
**section 태그를 가진 요소들을 전부 찾아서 가져와라**

---

## 4️⃣ `("section")`

### 의미

* 찾고 싶은 **HTML 태그 이름**
* 문자열 형태로 작성해야 함

```html
<section></section>
<section></section>
```

👉
이렇게 여러 개 있으면 **전부 가져옴**

---

## 5️⃣ `document.getElementsByTagName("section")`의 결과

이 코드는 **하나의 요소가 아니라**
👉 **HTMLCollection(배열 비슷한 것)** 을 반환함

예:

```js
[
  <section>...</section>,
  <section>...</section>
]
```

❗ 그래서 바로 `.innerHTML`을 못 씀
→ **몇 번째 section인지 선택해야 함**

---

## 6️⃣ `[0]`

### 의미

* 가져온 section 태그들 중 **첫 번째 것**
* 배열은 항상 **0부터 시작**

```js
[0] → 첫 번째 section
[1] → 두 번째 section
```

👉

```js
document.getElementsByTagName("section")[0]
```

=
**"첫 번째 section 태그 하나를 정확히 선택"**

---

## 7️⃣ `.innerHTML`

### 의미

* **선택한 요소 안의 HTML 내용을 의미**
* 읽을 수도 있고, 바꿀 수도 있음

### 예시

```html
<section>
  <p>안녕</p>
</section>
```

```js
section.innerHTML
```

→ `"<p>안녕</p>"`

---

### ✏️ 값을 대입하면?

```js
section.innerHTML = "<div></div>";
```

👉
section **안의 내용을 전부 지우고**
👉
새 HTML로 **통째로 교체**

---

## 8️⃣ `= html`

### 의미

* 오른쪽에 있는 `html` 변수의 내용을
* 왼쪽의 `innerHTML`에 **대입(assign)**

### 여기서 `html` 변수는?

```js
var html = "";
html += `<div class="bg1"></div>`;
html += `<div class="bg2"></div>`;
...
```

👉
**div들이 잔뜩 들어 있는 HTML 문자열**

---

## 🔥 전체 문장을 한국어로 완전 해석하면

> **"현재 웹페이지(document)에서
> section 태그들을 전부 찾아서
> 그중 첫 번째 section을 선택한 뒤,
> 그 안의 HTML 내용을
> html 변수에 들어있는 HTML 코드로 바꿔라"**

---

## 📌 아주 직관적인 의사코드로 바꾸면

```js
웹페이지에서 {
    section 태그들을 찾고
    첫 번째 section을 선택해서
    그 안에 html 변수에 만든 div들을 집어넣는다
}
```

---

## ❓ 질문에서 헷갈린 부분 바로잡기

> `innerHTML=html은 index.html에서 HTML 형태로 가져올거다(?)`

❌ **가져오는 게 아님**
✅ **JS에서 만든 HTML 문자열을 HTML에 넣는 것**

✔️ HTML ← JS 방향

---

## 🎯 clean() 함수도 같은 원리

```js
document.getElementsByTagName("section")[0].innerHTML = "";
```

👉
첫 번째 section 안의 내용을 **전부 삭제**

---


## 🔥 전체를 자연어로 다시 말하면

> **현재 웹페이지(document)에서
> 모든 section 태그를 찾고,
> 그중 첫 번째 section을 선택해서,
> 그 안의 HTML 내용을
> html이라는 변수에 들어 있는 값으로 바꾼다**

---

## 🧠 구조를 한 번에 이해하는 핵심 포인트

이 코드는 사실 **4단계 접근**이야:

1️⃣ **어디서?** → `document`
2️⃣ **무엇을?** → `section` 태그
3️⃣ **어느 것?** → `[0]`
4️⃣ **뭘 할 건데?** → `innerHTML = ...`

---

## 🚨 초보자가 제일 많이 헷갈리는 지점

### ❌ 오해

* document = HTML 태그
* getElementsByTagName = 하나만 가져온다

### ✅ 정정

* document = **페이지 전체**
* getElementsByTagName = **여러 개**
* `[0]`을 써야 하나를 집음

---

## 마지막으로 아주 중요한 한 줄

> **DOM 코드는 “외우는 문장”이 아니라
> “위치 → 대상 → 행동” 구조다.**
