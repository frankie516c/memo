<h3>2강.</h3>
<Vite>
리액트 파일(.jsx)를 브라우저가 해석할 수 없어서 표준인 Javascript(.js)로 변환함<br />
- 이걸 변환해주는 도구가 "Vite"<br />
- `Node.js`라는 Javascript 실행 환경 설치가 필수적<br />
<br />
프로젝트 설치 및 실행 명령어<br />
- `npm create vite@latest` 등의 명령어로 프로젝트를 생성하고, 필요한 라이브러리(패키지)들은 npm i (install)로 설치<br />
- `npm run dev`: 개발용 서버를 실행하는 명령어입니다. 코드를 수정하고 저장하면 브라우저에 즉시 반영(Hot Reloading)되어 개발이 편리
