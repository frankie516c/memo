우리 팀원 4명이 같은 DB 하나를 써야돼서 docker.desktop을 사용합니다.<br />
(개인 피씨에 DB 깔면 5개를 관리해야해서 이렇게 한다고 합니다.)<br />

npm이 node의 모듈이었다면 Docker Hub는 DB의 모듈이라고 생각하면 됩니다.<br />
검색창에 mariadb 검색하고 tag에 들어가서 12.1.2를 copy해서 터미널(cmd창)에 복붙합니다.<br />
(여러 버전이 있는데 어떤걸 다운받는지 정하는거는 하기 링크에서 MariaDB Server Version을 확인합니다.<br />
https://mariadb.org/download/?t=mariadb&p=mariadb&r=12.1.2&os=windows&cpu=x86_64&pkg=msi&mirror=blendbyte

제어창(?)에서 명령 프롬프트 열고 `docker pull mariadb:12.1.2` 를 설치합니다.<br />
<img width="1037" height="756" alt="image" src="https://github.com/user-attachments/assets/49bc881c-f361-472b-95e2-c8bc973d4ee4" />

이후에 데이터베이스 실행 명령어인 `docker run -d -p 3306:3306 --env MARIADB_ROOT_PASSWORD=1234 --name mariadb  mariadb:12.1.2`를 이어서 설치합니다.<br />
<img width="1041" height="620" alt="image" src="https://github.com/user-attachments/assets/103687ae-9206-4d9f-8ad4-07219c192e12" />

바탕화면에 `HeidSQL`을 킵니다. (건너뛰기 클릭)<br />
신규 클릭<br />
세션이름 `도커`<br />
암호 `1234` (위 db실행명령어에 password=1234로 되어 있기 때문)<br />
데이터베이스: 세미콜론으로 구분 -> 여기서 4가지가 뜨면 성공!<br />
열기 클릭<br />
