일차원 적인 답

```
if ( i == 1 ) console.log(i);
if ( i == 3 ) console.log(i);
if ( i == 5 ) console.log(i);
if ( i == 7 ) console.log(i);
if ( i == 9 ) console.log(i);
```

이 상태가 된 다음 줄이면 됨

플로우차트 사용 url: http://flowgorithm.org/
정보처리기사 자격증에도 이런 플로우차트가 쓰임
코딩이 어려우면 이 차트를 머리에 먼저 넣어야 됨
- 나중에 취업 할 때 **코딩테스트**도 이런 형식으로 이루어짐
- 내가 만든 알고리즘을 다양한 언어로 확인하는 기능이 있어 좋음
