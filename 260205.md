이미지 생성
dockerfile
```
FROM ubuntu:24.04

RUN apt update
RUN apt install -y nginx

EXPOSE 80

CMD [ "nginx", "-g", "daemon off;" ]
```

터미널 명령어:
dir (<- dockerfile이 뜨는지 확인 후 진행하기)
docker build -t web:0.1 .
docker images
docker run -d -p 80:80 --name web web:0.1
docker ps
