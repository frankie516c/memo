[method의 5가지 종류] (**암기**)
- get: read용으로 사용, 화면에서 바로 나타남 (조회)
- post: 데이터 조작 및 상세페이지로 보낼 대 사용 (등록)
- delete: background용 (삭제)
- put: background용 (수정)
- patch: background용(부분 수정/업데이트)

[method에 대하여]
- tmi: CRUD와 관련이 있어서 method의 종류가 5가지라고 함
- 실행해볼 것: `main.py`에서 5가지 method를 써본 후 `uv run fastapi dev` 실행시 아래 화면과 같이 나옴
- tmi: `uv run fastapi dev`로 했기 때문에 화면에 반영이 되는 것. `uv run fastapi run`으로 하면 반영이 안 됨
