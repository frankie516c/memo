<h3>직접 만들어보기1</h3>

문제
```
const 기호 = "■";
for( ; ; )
{
  
}
// 출력 예시 : ■■■■■■■■■■ 
```

답
```
    const 기호 = "■";
    let 값 = "";
    for ( let i = 0 ; i < 10 ; i++ )
    {
       값 = 값 + 기호;
    }
     console.log(값);
```
- `값 = 값 + 기호;`를 `값 += 기호;`로 써도 같음

<h3>직접 만들어보기2</h3>

문제
```
const 기호 = "■";
for( ; ; )
{
  
}
// 출력 예시 : 
// ■
// ■■
// ■■■
// ■■■■
// ■■■■■
// ■■■■■■
// ■■■■■■■
// ■■■■■■■■
// ■■■■■■■■■
// ■■■■■■■■■■
```

답
```
    const 기호 = "■";
    let 값 = "";
    for ( let i = 0 ; i < 10 ; i++ )
    {
       값 = 값 + 기호;
       console.log(값);
    }
```

내 오답
```
const 기호 = "■";
    let 값 = "";
    for ( let i = 0 ; i < 10 ; i++ )
    {
       console.log(기호);
       console.log("반복");
    }
    console.log(i);
```
내 실수
- `값 = 값 + 기호`를 넣지 않음
- `console.log("값")`을 넣지 않음
- `console.log("반복")`을 넣음
- `console.log("기호")`를 넣음

왜 이런 실수를 했는가?
- “반복하면 자동으로 결과가 쌓일 것”이라고 생각함
- 문자열은 직접 누적해서 저장해야 함
- for문 = 무조건 i < 10 같은 횟수 반복하는게 아님
- for문 = 초기화 / 조건 / 변화식을 자유롭게 설계하는 구조
