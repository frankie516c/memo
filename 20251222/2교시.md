```
//반복문 기본 문법
//1. 전역 변수
for(/* 2. 지역 변수 */ ; ; ) {
    console.log("반복");
}
```
- 1번은 밖에서도 가능
  - `var` 또는 `let`을 사용할 수 있음
  - `let`을 사용하는 것을 추천
      - why? 변하는 수면 안되고 정해진 값이어야 하기 때문
      - e.g.,
        ```
        let i = 0;
        ```
- 2번은 안에서만 사용 가능

```
let i = 0;
for( ; i < 1 ; ) {
    console.log("반복");
    i = 1;
}
console.log(i);
```
- 실행: `반복` 값 `1`
- why? i가 1보다 작을 때만 true이기 때문에 반복이 되는데 `i = 1`이라는걸 넣어졌으니까 1번 돌고 멈추게 될 것

```
let i = 0;
for( ; i < 1 ; ) {
    console.log("반복");
    i = i + 1;
}
console.log(i);
```
- `i = i + 1;`이게 **증감식**이기 때문에 세번째 콜롬에 넣어도 됨

```
let i = 0;
for( ; i < 1 ;  i = i + 1 ) {
    console.log("반복");
}
console.log(i);
```
**- 이렇게 for에 썼을 때 반복되는 공식이 더 쉽게 눈에 띄기 때문에 이렇게 쓰는 것을 추천한다**

```
for( let i = 0; i < 1 ;  i = i + 1 ) {
    console.log("반복");
}
console.log(i);
```
**- 같은 원리로 ` let i = 0`의 내용을 첫번째 콜롬에 넣음**

```
for( let i = 0; i < 3 ;  i = i + 1 ) {
    console.log("반복");
}
console.log(i);
```
- 이렇게하면 반복이 3번 됨
- 회전수를 제약하기 위해서는 두번째 콜롬을 수정하면 됨

Q. 개별의 값을 꺼내게 하기 코드를 완성하시오.
```
const a = [1,2,3,4,5];
for ( ; ; ) {
    console.log('배열${인덱스} : ${값});
}
```
A. 답
```
const a = [1,2,3,4,5];
for ( let 인덱스 = 0 ; 인덱스 < a.length ; 인덱스 = 인덱스 + 1) {
    console.log(`배열${인덱스} : ${ a[인덱스] }`);
}
```
답 안 봤을 때 한 나의 실수
- `인덱스`를 이용하지 않음 (i를 이용 - 이용하면 안되는건 아니지만 이렇게 할 경우 배열 뒤에도 i로 바꿔줘야 함)
- `a.[인덱스]` 을 값으로 사용하지 않음
- 두번째 콜롬에서 숫자를 대입하고 `a.length`를 사용하지 않음

답을 보고 나서도 한 나의 실수
- 백택(`)을 안 함
- a뒤에 점(.)을 넣음
