<h5>반복문</h5>
배열을 다룰 때 **반복문**을 쓴다<br />
왜? 배열 내에 있는 값 하나씩 꺼내거나 넣을 때 편리하게 쓰기 위해<br />

```
> for(;;) {
    console.log("반복")
}
```

- ;가 2개 있기 때문에 a;b;c 이렇게 세개의 변수가 있음<br />
- 변수는 index로 쓰임<br />
  - 첫번째 변수: 초기식 - 반복을 시작하기 전에 한 번만 실행<br />
      - 반복에 사용할 변수 선언<br />
          반복문을 위한 변수이기 때문에 사용이 불가능하다는 뜻
      - 초기값 설정<br />
  - 두번째 변수: 조건식 - 매 반복마다 검사<br />
      - `true`면 반복<br />
      - `false`면 종료<br />
      - 사실상 반복을 멈출지 마맂 결정하는 핵심<br />
  - 세번째 변수: 증감식 - 첫번째 변수를 증감시킬 것인지<br />
      - 각 반복이 끝날 때마다 실행
      - 변수 증가/감소
      - 다음 상태로 이동
- 무한반복 함<br />
  - 멈춰주고 싶을 때 

```
> var i = 0;
  for (;i == 0;) {
      console.log("반복");
      }
```

  i 값이 바뀌지 않아서 얘도 무한반복

그렇다면 `값이 거짓`일 때 무한반복을 하지 않음
```
> var i = 0;
  for (;i == 1;) {
      console.log("반복");
      }
```
`논리`를 넣어줘서 무한반복을 멈춰주는 반복문 기본 문법
```
> var i = 0;
  for (;false;) {
      console.log("반복");
      }
```
```
> var i = 0;
  for (;true;) {
      console.log("반복");
      }
```
true라고 하면 무한반복을 ㅎ ㅐ줌
