# ✅ for문이란?

**같은 코드를 여러 번 반복해서 실행할 때 사용하는 문법**이에요.

예:

```javascript
console.log("안녕");
console.log("안녕");
console.log("안녕");
```

⬇️ for문으로 줄이면

```javascript
for (let i = 0; i < 3; i++) {
    console.log("안녕");
}
```

---

# ✅ 기본 for문 구조 (가장 중요 ⭐)

```javascript
for (초기값; 조건식; 증감식) {
    실행할 코드
}
```

### 예제

```javascript
for (let i = 0; i < 3; i++) {
    console.log(i);
}
```

---

# ✅ for문 3요소 쉽게 이해하기

## 1️⃣ 초기값

```javascript
let i = 0;
```

* 반복을 시작할 **기준 값**
* 보통 `0`부터 시작
* 딱 **한 번만 실행**

👉 “i를 0부터 시작할게”

---

## 2️⃣ 조건식

```javascript
i < 3
```

* 반복을 **계속할지 말지 결정**
* `true` → 계속 반복
* `false` → 반복 종료

👉 “i가 3보다 작으면 반복해”

---

## 3️⃣ 증감식

```javascript
i++
```

* 반복이 끝날 때마다 실행
* 보통 `i`를 1씩 증가

👉 “한 번 실행할 때마다 i를 1 올려”

---

# ✅ 실행 순서 (시험에도 중요!)

```javascript
for (let i = 0; i < 3; i++) {
    console.log(i);
}
```

### 실행 흐름

1. `i = 0` (초기값, 1번)
2. `i < 3` 확인
3. 코드 실행 (`console.log(i)`)
4. `i++`
5. 다시 조건 확인
6. 조건이 false면 종료

---

# ✅ 출력 결과

```
0
1
2
```

---

# ✅ 자주 쓰는 for문 예제

## 1️⃣ 1부터 5까지 출력

```javascript
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```

---

## 2️⃣ 0부터 9까지 출력

```javascript
for (let i = 0; i < 10; i++) {
    console.log(i);
}
```

---

## 3️⃣ 10부터 1까지 거꾸로

```javascript
for (let i = 10; i >= 1; i--) {
    console.log(i);
}
```

---

# ✅ for문에서 꼭 기억할 포인트 ⭐

✔️ 반복 횟수가 **정해져 있을 때** 사용
✔️ 조건식이 `false`가 되면 바로 종료
✔️ `i++` 안 쓰면 **무한 반복** 될 수 있음
✔️ `i`는 보통 **반복 횟수 역할**

---

# ✅ 초보자가 많이 헷갈리는 실수 ❌

❌ 조건식 잘못 쓰기

```javascript
for (let i = 0; i = 3; i++) { } // ❌
```

⭕ 올바른 조건식

```javascript
for (let i = 0; i < 3; i++) { } // ⭕
```

---

# ✅ 한 줄 요약 📌

> **for문은 “몇 번 반복할지 정해져 있을 때 쓰는 반복문”이고,
> 초기값 → 조건 → 실행 → 증가 순서로 동작한다.**
