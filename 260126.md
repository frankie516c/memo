[method의 5가지 종류] (**암기**)
- get: read용으로 사용, 화면에서 바로 나타남 (조회)
- post: 데이터 조작 및 상세페이지로 보낼 대 사용 (등록)
- delete: background용 (삭제)
- put: background용 (수정)
- patch: background용(부분 수정/업데이트)

[이미지 추가 작업]
1) `images` 폴더를 만들고 폴더 안에 아무 이미지를 넣음
2) `main.py`에 하기 코드 추가
```
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
import os

app = FastAPI()

static_dir = os.path.join(os.path.dirname(__file__), "images")
app.mount("/images", StaticFiles(directory=static_dir), name="images")
```
checkpoint: `app.mount("/images," <- 여기에 오는 이미지는 주소이기 때문에 `/images`가 아니어도 됨

3) `uv run fastapi dev` 실행
4) 하기 이미지 경로로 들어가서 주소에 다음과 같이 기입 `http://127.0.0.1:23306/images/0.png`
