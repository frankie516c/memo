[method의 5가지 종류] (**암기**)
- get: read용으로 사용, 화면에서 바로 나타남 (조회)
- post: 데이터 조작 및 상세페이지로 보낼 대 사용 (등록)
- delete: background용 (삭제)
- put: background용 (수정)
- patch: background용(부분 수정/업데이트)

[method에 대하여]
- tmi: CRUD와 관련이 있어서 method의 종류가 5가지라고 함

[main.py 간략화] - 기능 넣기 전 url path 관리 방법

1) `root.py`를 만들어준다

```
from fastapi import APIRouter

main = APIRouter(prefix="/user", tags=["연습용"])

@main.get("")
def read_root():
    return {"Hello": "World"}

@main.post("")
def read_root():
    return {"Hello": "World"}

@main.put("")
def read_root():
    return {"Hello": "World"}

@main.delete("")
def read_root():
    return {"Hello": "World"}
```

2. main.py는 다음과 같이 간략화 한다

```
from fastapi import FastAPI
from fastapi.staticfiles import StaticFiles
import os
from root import main

app = FastAPI()

static_dir = os.path.join(os.path.dirname(__file__), "update")
app.mount("/update", StaticFiles(directory=static_dir), name="update")

app.include_router(main)

@main.get("/test")
def read_root():
    return {"Hello": "World"}
```
