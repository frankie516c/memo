```
git clone (repositories 주소 복사)
```

- 완료 후 경로 다시 확인(밖으로 나가있음) 날짜파일로 경로
```
cd 2601xx
```
**Issue에서 내 branch를 만들어야 함 (항상 새로 하는 작업임)**

- 이동 후 내 branch 복사
```
git fetch origin
git checkout 조윤주
```

- 내껄로 잘 들어왔는지 확인 `* 조윤주`가 떠야 됨
```
git branch
```
- 이후에 오늘 날짜 폴더에 아래 라우터 설치
```
npm i react-router
```

내가 pull하려면
```
git pull
```

내가 가져오려면 (push하려면)
```
git add .
git commit -m "내용"
git push
```
