#폼태그 만들기
```HTML
<!DOCTYPE html>
<html lang="ko">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content=""width=device-width, initial-scale="1.0">
	<link rel="icon" jref="../../images/logo.png"/>
	<title>폼 알아보기</title>
</head>
<body>
    <form> 
       <input type="text" name="txt" required placeholder="글 작성해주세요"/> 
       <!-- <input type="submit" /> -->
       <button type="submit">제출</button>
    </form>
</body>
</html>
```
- method="post" 얘가 빠져야 제출을 눌렀을 때 page not found가 안 뜸
- 13번: 맨 앞에 "아이디 : "라고 붙이면 박스 앞에 아이디라는 글자가 보임
- 14번과 15번이 같은 역할을 함 (택1 쓰면 됨)
- get방식은 아이디 비번이 노출돼서 post로 씀-보안이슈
- name=txt가 정의가 되어있어야 함. 얜 input이 갖고 있는 변수를 뜻함
- required가 있어야 빈칸 상태에서 제출 할 때 입력하라는 안내문이 뜸
- input submit이나 button type 둘 중 하나만 쓰기
