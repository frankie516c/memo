
  강사님 코드
  ```HTML
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
  <style>
    * {box-sizing: border-box;}
    html,body {margin: 0; padding: 0;}
    form {padding: 10px 30px;}
    div {display: flex; padding: 5px 0;}
    label {width: 50px;}
    input {width: calc(100% - 50px);}
    button {width: 100%; padding: 5px 0; border: 0; border-radius: 5px; background-color: #5865f2;}
  </style>
</head>
<body>
  <form>
    <div>
      <label for="">글자:</label>
      <input type="text" name="txt" id="txt" required placeholder="글자를 입력해주세요." />
    </div>
    <div>
      <label for="num">숫자:</label>
      <input type="number" name="num" id="num" required placeholder="10부터 입력해주세요." min="10" max="100" step="10" />
    </div>
    <div>
      <label for="file">파일:</label>
      <input type="file" name="file" id="file" />
    </div>
    <button type="submit">업로드</button>
  </form>
</body>
</html>
```
- form을 넣어야 여백이 생긴다
- `div`를 `display: flex`해야 된다
- `input`의 너비를 작성하기 위해서 `cal(100% - 50px)`이렇게 계산해줘야한다
- 글자가 들어가는건 `txt`
- 숫자가 들어가는건 `num`
- 파일이 들어가는건 `file`
**- "선택된 파일 없음"이라는 문구를 내가 직접 넣는 것이 아닌 파일의 속성 때문에 자동으로 생기는 문구**
